# DE BLE Kitchen Scale

Интеграция для Home Assistant, добавляющая поддержку кухонных весов Nutridays KT630LB через Bluetooth.

![KT630LB Scale](custom_components/de_ble_kitchen_scale/brand/icon.png)

## Поддерживаемые устройства

- **Модель:** KT630LB (кухонные весы)
- **Производитель:** Nutridays
- **Bluetooth-сервис:** `0000ffb0-0000-1000-8000-00805f9b34fb`

## Возможности

- Автоматическое обнаружение весов через Bluetooth
- Отображение веса в граммах
- Сенсор статуса подключения
- Автоматическое подключение при появлении весов
- Энергосберегающий режим (отключение при отсутствии весов)

## Установка

### Через HACS (рекомендуется)

1. Убедитесь, что у вас установлен [HACS](https://hacs.xyz/)
2. Откройте HACS в Home Assistant
3. Нажмите на три точки в правом верхнем углу
4. Выберите "Пользовательские репозитории"
5. Добавьте:
   - **URL:** `https://github.com/de-andrei/de_ble_kitchen_scale`
   - **Категория:** `Integration`
6. Нажмите "Добавить"
7. Найдите "DE BLE Kitchen Scale" в списке интеграций HACS
8. Нажмите "Скачать"
9. Перезапустите Home Assistant

### Ручная установка

1. Скачайте последний релиз из [репозитория](https://github.com/de-andrei/de_ble_kitchen_scale/releases)
2. Распакуйте папку `custom_components/de_ble_kitchen_scale` в директорию `config/custom_components/` вашего Home Assistant
3. Перезапустите Home Assistant

## Настройка

1. Перейдите в **Настройки** → **Устройства и службы**
2. Нажмите **"Добавить интеграцию"**
3. Найдите **"DE BLE Kitchen Scale"** в списке
4. Выберите ваши весы KT630LB из списка обнаруженных устройств
5. Нажмите "Отправить"

## Сенсоры

После настройки будут созданы следующие сенсоры:

| Сущность | Имя | Описание |
|----------|-----|----------|
| `sensor.kitchen_scale_weight` | Weight | Текущий вес в граммах (всегда 0, если нет измерения) |
| `sensor.kitchen_scale_connection_status` | Connection Status | Статус подключения к весам |
